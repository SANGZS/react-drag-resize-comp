# React Test

本项目是一个基于 React 的前端演示项目，包含多个可通过路由切换的组件示例。

## 功能简介
- 通过路由预览不同组件（如 Demo、EnhancedDemo）
- 支持自定义端口号（已配置为 3000）

## 安装与使用

1. 克隆项目：
   ```bash
   git clone <你的仓库地址>
   cd react-test
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
   默认访问地址：http://localhost:3000

## 目录结构
- `src/components/`：主要组件目录
- `src/App.tsx`：主入口，包含路由配置
- `vite.config.ts`：Vite 配置文件
